Colours

Dragon CSS has a set of standard colour classes that can be used to set the colour of the background or text.

To change the background, simply add the name of the colour, for example a class of "red" will set the background colour to red.

To change the colour of the text, just add a class of colour-text, for example "maroon-text" will set the text colour to maroon.

Adding a modifier class of "pastel","classic" or "neon" will change the shade of the colour. For example "neon yellow-text" will set the text colour to neon yellow.
