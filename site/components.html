Components

Dragon CSS includes some useful classes to make styling common components easier.

Tabs

Tabs can be created by adding a class of "tabs" to a an unordered list of links.

The colour of tabs and the hover colour can be set in settings.

Adding a class of active will add a subtle highlight to that tab.
