Typography

Headings

Headings are automatically styled but you can apply the same styles using the classes "h1","h2","h3","h4","h5" and "h6".

.h1 Heading 1

.h2 Heading 2

.h3 Heading 3

.h4 Heading 4

.h5 Heading 5
.h6 Heading 6

Display

Use a class to create display text.

.display-l for large display text

.display-m for medium sized display text

.display-s for small display text

Text Size

Use a class to set the font-size of an element.

.text-xl for huge text

.text-l for big text

.text-m for normal-sized text

.text-s for small text

.text-xs for teeny tiny text

Fonts

You can apply a particular font to an element by just adding the relevant class.

.serif-font

.sans-font

.mono-font

.heading-font

.fancy-font

.slab-font

.brand-font

Helper Methods

.text-left

Aligns text to the left

This text has a class of "text-left"

.text-center

Aligns text to the center

This text has a class of "text-center"

.text-right

Aligns text to the right

This text has a class of "text-right"

.justify

Justifies text

This text has a class of "justify". This will space the text out so that it always aligns to the left and the right sides.

.bold

Makes text bold

This text has a class of "bold"

.italic

Makes text italic.

This text has a class of "italic"

.underline

Underlines

This text has a class of "underline"

.no-underline

Ensures that text is not under-lined.

This text has a class of "no-underline", even this anchor tag.

.line-through or .strike-through

Strikes a line through the text.

This text has a class of "line-through" or "strike-through"

.uppercase

Makes all the text uppercase.

This text has a class of "uppercase"

.lowercase

Makes all the text lowercase.

This text has a class of "lowercase"

.capitalise or .capitalize

Makes every word start with a capital letter.

This text has a class of "capitalise"

.line-height

Gives the text a line-height of 1.

This text has a class of "line-height-1"

.bold

Gives the text a line-height of 1.5.

This text has a class of "line-height-1-5"

.italic

Gives the text a line-height of 2.

This text has a class of "line-height-2"
