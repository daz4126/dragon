Grids

Dragon has a simple grid system that allows you to create complex but repsonsive layouts with just a few classes.

Basic Grid

Adding a class of "grid" to a containing element will create a 3-column grid.

No other classes are needed.

The default number of columns can be changed in settings.

Column Spans

You can make an element span more columns by using the class "col-span-n", where n is the number of columns you want it to span.

For example, the first column has a class of "col-span-2" and the column in the bottom row has a class of "col-span-3".

Starting Column

You can make an element start in a specific column by using the class "start-n", where n is the number of the column you want it to start in.

For example, the first column has a class of "start-2" and the last element has a class of "start-3".

Gaps
