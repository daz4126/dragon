Utility Classes

Dragon CSS includes some nifty utility classes to help tweeak the layout and presentation.

Add any of these classes to an element to style it.

.float-end or .float-right

Floats an element to the right

.float-start or .float-left

Floats an element to the left

inline, inline-block, block, absolute, relative, sticky, badge, container-s, container-m, no-bullet, horizontal-list etc

Jusitifies text

.full-screen

Makes the element fill the screen in the horizontal and vertical direction.

.full-width

Makes the element fill the full width of the screen.

.align-vertical

Aligns all the child elements vertically

.align-horizontal

Aligns all the child elements vertically

.align-center

Aligns child elements to the center

.align-right

Aligns child elements to the right

.align-left

Aligns child elements to the left

.fixed
Makes the element fixed position
.no-padding
Removes all padding
.no-margin
Removes all margins
.pointer
Makes a pointer appear when the cursor hovers over the element
.circle
Makes the element circular
.clearfix
Clears any floated elements that come before the element
.hidden
Hides the element from view
.hide-xs
Hides the element from view on tiny screens
.hide-sm
Hides the element from view on small screens or smaller
.hide-md
Hides the element from view on medium-sized screens or smaller
.hide-lg
Hides the element from view on large screens or bigger
.hide-print
Hides the element from view when printed

.dropshadow

Adds a subtle drop shadow to the element

.muted

Makes the element appear muted by increasing opacity
