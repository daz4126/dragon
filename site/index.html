Beautifully Simple CSS

Features

  • Classless styles give you a beautiful and responsive site out of the box
  • Easy to customize using CSS variables
  • A variety of colourful backgrounds, text and borders that can be easily changed with classes
  • Simple but powerful grid layout system
  • Simple components such as tabs, cards and tags can be created using a single class
  • Utility classes to help tweak the layout and presentation
  • Check out the demo page to see what it can do

Get Started

All you need is one CSS file, dragon.min.css, linked to your page

You can download it here or just link directly to the CDN: 

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/daz4126/dragon@latest/dragon.min.css">

Dragon CSS has been inspired by the following CSS frameworks:
